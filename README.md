# Что это такое ?

Простой инструмент для преобразования русского(кирилица) текста в латинские символы.

## Возможности

- Простой и минималистичный интерфейс
- Мгновенный перевод при вводе

## Пример работы
```mermaid
graph LR
A[съешь ещё этих мягких французских булок, да выпей чаю] -- Преобразуется в  --> B(cbeIIIb_eIIle_eTux_M9IrKux_FpaHuy3cKux_6yJI0K, _ga_BbIneu_4aIO)

```
